Tisbe 테마 설명

티스베(Tisbe)테마는 Tiny is Best에서 따온 약자이며 이름처럼 작은 조각들을 모아 하나의 모양을 만듭니다.
애니아 홈페이지를 만들면서 쌓아온 야매기술 모두 모았습니다.
대부분 서버지기의 귀차니즘을 반영하여 약자로 표시합니다.
서버지기가 사용하지 않는 태그는 지원하지 않습니다. ex) table
동적사이트보다 정적사이트 알맞게 설게되었습니다.
w2 = width(100% / 12* 2)
p10 = padding:10px;
m5 = margin:5px;
퍼즐처럼 이런 작은 단위들을 조합하여 하나의 모양을 생성합니다.

색상의 종류

미리 정의되어있는 색상을 선택해 사용하실 수 있습니다.

테마에 기본이 되는 색들 입니다.
티스베에 존재하는 고급색상입니다. 플렛/파스텔톤의 색상들이 있습니다.
이후에 계속 추가됩니다.
c201xx 형식의 색상은 pantone이 선정한 해당년도의 대표색상입니다.
위에 존재하는 모든 색상은 아래 나열된 지정색상에서 사용하실 수 있습니다.
<printc-{color}>을 작성하시면 해당 색상의 HEX값을 출력합니다. ex) <printc-blue>

버튼의 색상 지정

다양한 형태의 버튼의 양식이 정의되어있습니다.

버튼의 예제입니다. class="btn {color}" 형태로 적용하실 수 있습니다.
기본적으로 배경 글자색이 흰색으로 지정됩니다. 단, 흰색의 경우 글자는 검정색으로 변경됩니다.
radi0~20 를 통해 border-radius값을 설정할 수 있습니다. (짝수단위만 존재합니다.)
radi는 !important가 적용된 속성입니다.
/sass/variable.scss 파일의 default-btn-radius 값을 변경후 컴파일하면 기본 버튼의 radius값을 결정할 수 있습니다.
만약 흰배경에 글자에 색상을 넣고 싶을경우 o 를 추가하시면 됩니다.
left, top, bottom, right으로 버튼의 특정 방향을 강조해 나타낼 수 있습니다.
